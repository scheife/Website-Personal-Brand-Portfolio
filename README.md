# Personal Brand Portfolio - Alexander Kai Scheiflinger

Servus! Das ist der Quellcode für meine persönliche Portfolio-Website. 
Die Seite dient als digitale Visitenkarte und präsentiert meine Fähigkeiten in den Bereichen Logistik-Management, Handwerk und mein wachsendes Interesse an moderner Web-Technologie.

## Tech Stack & Features
- **Frontend:** HTML5, CSS3, JavaScript
- **Icons:** Boxicons
- **Animationen:** Scroll-Reveal & CSS-Keyframes
- **Kontaktformular:** Integriert via Formspree.io
- **Responsive Design:** Optimiert für Desktop, Tablet und Smartphone

## Über das Projekt
Dieses Projekt basiert auf einem modernen Dark-Mode Design. Ich habe das ursprüngliche Template angepasst, um meine "Macher-Mentalität" widerzuspiegeln – eine Brücke zwischen handwerklicher Präzision, logistischer Führung und technischer Neugier.
